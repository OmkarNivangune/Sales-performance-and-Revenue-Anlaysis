# Sales Performance & Revenue Analysis

##  Overview
This project analyzes sales data to track revenue, identify trends, and generate insights using interactive dashboards.

## Objectives
* Monitor sales and revenue performance
* Identify top products and regions
* Analyze monthly trends
* Track key business KPIs

## Tools Used
* Power BI
* SQL
* Python (Pandas)
* Excel

## Workflow
* Data Cleaning & Transformation (ETL)
* Exploratory Data Analysis (EDA)
* Dashboard Creation in Power BI

## Key Features:
* Revenue & Sales Overview
* Top Products Analysis
* Monthly Trends
* Region-wise Performance
* KPI Dashboard


## Insights:
* Found top revenue-generating products
* Identified sales trends and patterns
* Highlighted high-performing regions
